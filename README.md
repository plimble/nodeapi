# Nodejs bootstrap project

### Installation
```
npm install
```

### Usage
dev mode

```
npm run dev
```

production
need to precompile to for gain performance

```
npm run build
npm run start
```

### Feature
- Babel ES6
- eslint
- typecheck
- Async with generator or async and await (transform to coroutine and blubird)
- bottlejs for inversion of control
